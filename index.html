Introduction

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It provides the structure for content on the web, such as text, images, links, and multimedia.

HTML is not a programming language; rather, it is a markup language that defines how content is displayed in the browser.

Browsers read HTML documents and render them into visual or interactive web pages. Without HTML, there would be no structured web.

Example of a Simple Web page


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
Elements

An HTML element is the building block of a webpage. It usually consists of an opening tag, content, and a closing tag.

For example, the <p>element represents a paragraph: 


<p>This is a paragraph.</p>

There are three major types of HTML elements:

  • Block-level elements: Take up the full width of the page. Example: <div>, <p>, <h1>.
  • Inline elements: Take up only as much space as their content. Example: <span>, <a>, <em>.
  • Empty elements: Do not have closing tags. Example: <img>, <br>, <hr>.

Understanding how elements work is the foundation of creating structured documents.

Attributes

Attributes give additional information about elements. They are always placed inside the opening tag and usually follow a name="value" format.

        
<img src="photo.jpg" alt="A beautiful frame" width="300">

In the example above:

  • src specifies the image source
  • alt provides alternative text for accessibility.
  • width adjusts the image size.

Common attributes include:

  • id - unique identifier for an element.
  • style - allows inline CSS styling.
  • href - used in links to point to a URL.

Attributes make HTML elements more powerful and descriptive.

Headings

Headings are used to define the structure of a webpage and give hierarchy to content. HTML provides six levels of headings, from <h1> (the most important) to <h6> (the least important).

Example of HTML headings:

        
<h1>Main Title of the Page</h1>
<h2>Subsection Title</h2>
<h3>Sub-subsection Title</h3>
<h4>Fourth Level Heading</h4>
<h5>Fifth Level Heading</h5>
<h6>Sixth Level Heading</h6>

Key points about headings:

  • <h1> is typically used for the main title of the page.
  • <h2> to <h6> are used for subsections.
  • Headings help both users and search engines understand the structure of your content.
  • Only one <h1> should be used per page for best SEO practices.
Comments

HTML comments allow you to add notes or explanations inside your HTML code that are not displayed in the browser. They are very useful for:

  • Explaining sections of your code for yourself or other developers.
  • Temporarily disabling parts of code without deleting them.
  • Leaving reminders or instructions within your HTML files.

Syntax of HTML comments:

        
<!--This is a comment-->

Example : 

        
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
    <!-- This is the head section -->
  </head>
  <body>
    <h1>Welcome to My Page</h1>
    <!-- Main heading above -->
    <p>This is a paragraph.</p>
    <!-- Paragraph ends here -->
  </body>
</html>

Good practice: Use comments to explain complex code or sections, especially in larger projects.
